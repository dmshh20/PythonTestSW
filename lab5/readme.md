Короткі відповіді на завдання:
- Чому при передачі None створюється об'єкт Anonymous?
  Якщо name == None, конструктор викликає клас-метод anonymous_user(), який повертає об'єкт з ім'ям "Anonymous".

- Як змінити текст привітання при виклику say_hello()?
  Викликати метод з аргументом, наприклад: MyName.say_hello("Привіт усім!") або instance.say_hello("Ваше повідомлення").

- Додано метод name_length(), що рахує кількість букв імені (використовується len()).

- Чому різниця в кількості імен?
  У прикладі ми будуємо словник {name: MyName(name) for name in names}. Якщо у кортежі names є дублікати або None, ключі словника можуть перезаписатись — тому len(names) і len(created_objects) можуть відрізнятись.

- Перетворення імені:
  Конструктор гарантує, що ім'я починається з великої літери (.capitalize()).

- Валідація:
  Якщо ім'я містить цифри або символи, піднімається ValueError("Ім'я може містити лише літери!").

- Метод create_email(domain="itcollege.lviv.ua") дозволяє змінювати домен після @.

- Властивість full_name:
  Повертає рядок у форматі: "User #<id>: <name> (<email>)".

- Метод save_to_file(filename="users.txt"):
  Додає рядок з full_name в кінець файлу.
