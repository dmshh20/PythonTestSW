# Work with Classes — Lab5

Мета: виконати всі комірки в classes.ipynb, додати нові комірки з виконаними додатковими завданнями (кожна секція має кодову ячейку з прикладом).

Як працювати:
1. Перейдіть в папку:
   cd d:\pythonTestSW\PythonTestSW\lab5\work-with-classes
2. Відкрийте ноутбук:
   jupyter notebook classes.ipynb
3. Виконайте всі ячейки (Run → Run All) або по одній. У ноутбуку вже додано виконані приклади та пояснення.

Що включено:
- Інкапсуляція: BankAccount + випадкові виклики deposit/withdraw у циклі.
- Наслідування: Vehicle → Car з додатковим методом у Vehicle.
- Поліморфізм: Animal, Dog, Cat, Fish (демонстрація поведінки Fish).
- Абстракція: Shape (abc) та Circle.
- Ігрова симуляція: Item, Sword, Axe, Bow (Bow має range_power і reload), коротка покрокова демонстрація.

Скопіюйте результати в звіт (.md) або зробіть скриншоти.
